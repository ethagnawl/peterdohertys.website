# Source code for peterdohertys.website

This project uses the Haskell static site generator, [Hakyll](https://jaspervdj.be/hakyll/).

## Build

`stack setup && stack build && stack exec site build`

## Develop

`stack exec site watch`

## Deploy

`stack exec deploy`
